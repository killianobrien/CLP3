Section 4.7 Impulses and steps: The Dirac and Heaviside functions

Some applications generate differential equations featuring discontinuous function. In this section we show how such functions can be represented and dealt with by the Laplace transform solution method.

For example, if \(\mathrm{i}(t)\) is the current in the electrical circuit shown in figure 4.7.1, then \(\mathrm{i}(t)\) satisfies the integral equation

\begin{equation*} L \frac{di}{dt} + Ri + \frac{1}{C} \int_0^1 i(t) \, dt = E_0 \sin(\omega t). \end{equation*}

If the circuit is switched on for a certain time period then the graph of the current might be shown by figure 4.7.2, or for a direct current (a battery say) powering the circuit, like that in figure 4.7.3.

Such functions, which are switched on in a certain time interval, and then switched off at other times, can be expressed with the aid of the Heaviside step function.

Definition 4.7.4. Heaviside step function.

The Heaviside step function \(H(t)\) is defined as

\begin{equation*} H(t) = \begin{cases} 0 \amp\text{if } t\lt 0 \\ 1 \amp\text{if } t \geq 0\end{cases} . \end{equation*}

Its graph is shown in figure 4.7.5. We think of it as representing a signal of strength \(1\) which is switched off until \(t=0\) and then switched on thereafter.

One can easily have the signal switch on at other points apart from \(t=0\text{,}\) by considering the Heaviside function where the argument is shifted by a fixed amount. Consider the function \(H(t-a)\) which satisfies

\begin{equation*} H(t-a) = \begin{cases} 0 \amp \text{if } t\lt a \\ 1 \amp \text{if } t \geq a\end{cases} , \end{equation*}

and whose graph is shown in figure 4.7.6.

Using combinations of different Heaviside functions like this we can produce arbitrary signals that are switched on in a desired time interval and switched off at other times, as in the following examples.

Example 4.7.7. Applications of the Heaviside step function.

  1. Let \(a \lt b\text{.}\) What are the various values taken on by the function \(g\) defined by

    \begin{equation*} g(t) = H(t-a) - H(t-b). \end{equation*}

    Draw the graph of \(g\text{.}\)

  2. Let \(a \lt b\text{.}\) What are the various values taken on by the function \(f\) defined by

    \begin{equation*} f(t) = \big ( 1 + \cos(t) \big ) \big ( H(t-a) - H(t-b) \big ). \end{equation*}

    Draw the graph of \(f\text{.}\)

  3. Consider the function \(V\) defined by

    \begin{equation*} V(t) = \begin{cases} 0 \amp \text{if } t \lt 0 \\ t \amp\text{if } 0 \leq t \lt 2 \\ 2 \amp \text{if } t \geq 2 \end{cases}. \end{equation*}

    Draw a graph of \(V\) and find a definition for \(V\) (not utilising different cases) involving suitable Heaviside step functions.

Its important to remember that a function \(\phi(t) \big ( H(t-a) - H(t-b) \big )\) is the signal \(\phi(t)\) switched on between the values \(t=a\) and \(t=b\) and switched off at other times.

We also want the ability to model impulses, which is done with the Dirac delta function.

Definition 4.7.8. Dirac delta function.

The Dirac delta function \(\delta\) is the function 1  defined by

\begin{equation*} \delta(t) = \begin{cases} \infty \amp\text{if } t=0 \\ 0 \amp\text{if } t \neq 0 \end{cases}. \end{equation*}

The position of the spike can be moved, just as we moved the step in the Heaviside function earlier, by shifting the argument, i.e. the function \(\delta(t-a)\) satisfies

\begin{equation*} \delta(t-a) = \begin{cases} \infty \amp\text{if } t=a \\ 0 \amp\text{if } t \neq a\end{cases}. \end{equation*}
Strictly speaking this is not a real valued function in the usual sense, it is more properly called a distribution

The transforms of the functions \(\delta\) and \(H\) are given by the following results.

Theorem 4.7.9. Transform of the Heaviside.

The Laplace transform of the Heaviside step function \(H(t-a)\) is given by

\begin{equation*} \L{H(t-a)} = \frac{e^{-as}}{s} . \end{equation*}
Proof.

In lecture.

Theorem 4.7.10. Transform of products with Heaviside.

If

\begin{equation*} \L{f(t)} = F(s) \end{equation*}

then the Laplace transform of the function \(f(t-a)H(t-a)\) is given by

\begin{equation*} \L{f(t-a)H(t-a)}= e^{-as} F(s) . \end{equation*}
Proof.

In lecture.

Theorem 4.7.11. Transform of the Dirac delta function.

The Laplace transform of the Dirac delta function \(\delta(t-a)\) is given by

\begin{equation*} \L{\delta(t-a)} = e^{-as} . \end{equation*}
Proof.

In lecture.

These results can be applied to find transforms and inverse transforms of more general functions involving \(H\) and \(\delta\text{.}\)
