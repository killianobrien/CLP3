Definition 5.5.2. Uniform convergence.

A sequence of functions \(f_n:\mathbb{R} \to \mathbb{R}\) converges uniformly to a limit function \(f:\mathbb{R} \to \mathbb{R}\) across the whole domain \(\mathbb{R}\) if and only if for each argument \(a \in \mathbb{R}\) the sequence of values \(\left \{ f_n(a) \right \}_{n=1}^\infty\) converges to the limit value \(f(a)\) and moreover there is an overall bound on the rate that this convergence takes place at. Put formally, this is, for all \(\epsilon \gt 0\) there exists \(N \in \mathbb{N}\) such that for all \(a \in \mathbb{R}\text{,}\) \(|f_n(a) - f(a)| \lt \epsilon\text{.}\)
