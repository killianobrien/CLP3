A function \(f:\mathbb{R} \to \mathbb{R}\) is called even if and only if

\begin{equation*} \forall x \in \mathbb{R}, \, f(-x)=f(x). \end{equation*}

A function \(f:\mathbb{R} \to \mathbb{R}\) is called odd if and only if

\begin{equation*} \forall x \in \mathbb{R}, \, f(-x)=-f(x). \end{equation*}
