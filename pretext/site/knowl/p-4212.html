Find the Laplace transform of the function \(R\) defined by

\begin{equation*} R(t) = t^2 H(t-2). \end{equation*}

Hint: For this it is necessary to re-express \(R\) as a combination of functions, all of which have the form \(f(t-a)H(t-a)\text{,}\) so that we can apply theorem 4.7.10.
