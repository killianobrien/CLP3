Definition 5.4.1. Odd and even functions.

A function \(f:\mathbb{R} \to \mathbb{R}\) is called even if and only if

\begin{equation*} \forall x \in \mathbb{R}, \, f(-x)=f(x). \end{equation*}

A function \(f:\mathbb{R} \to \mathbb{R}\) is called odd if and only if

\begin{equation*} \forall x \in \mathbb{R}, \, f(-x)=-f(x). \end{equation*}

The graph of an even function is symmetric under reflection in the \(y\)-axis. Whereas the graph of an odd function is symmetric under rotation about the origin by \(\pi\) radians.
