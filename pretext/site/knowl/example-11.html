Example 1.7.7. Applications of the Heaviside step function.

  1. Let \(a \lt b\text{.}\) What are the various values taken on by the function \(g\) defined by

    \begin{equation*} g(t) = H(t-a) - H(t-b). \end{equation*}

    Draw the graph of \(g\text{.}\)

  2. Let \(a \lt b\text{.}\) What are the various values taken on by the function \(f\) defined by

    \begin{equation*} f(t) = \big ( 1 + \cos(t) \big ) \big ( H(t-a) - H(t-b) \big ). \end{equation*}

    Draw the graph of \(f\text{.}\)

  3. Consider the function \(V\) defined by

    \begin{equation*} V(t) = \begin{cases} 0 \amp \text{if } t \lt 0 \\ t \amp\text{if } 0 \leq t \lt 2 \\ 2 \amp \text{if } t \geq 2 \end{cases}. \end{equation*}

    Draw a graph of \(V\) and find a definition for \(V\) (not utilising different cases) involving suitable Heaviside step functions.
