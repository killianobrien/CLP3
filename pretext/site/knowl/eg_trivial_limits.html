Example 2.1.4.
  1. If \(f(x,y)\) is the constant function which always takes the value \(L\text{,}\) then
    \begin{equation*} \lim_{(x,y)\to(a,b)} f(x,y) = L \end{equation*}
  2. If \(f:\bbbr^2\rightarrow\bbbr^2\) is defined by \(f(x,y) = (x,y)\text{,}\) then
    \begin{equation*} \lim_{(x,y)\to(a,b)} f(x,y) = (a,b) \end{equation*}
  3. By definition, as \((x,y)\) approaches \((a,b)\text{,}\) \(x\) approaches \(a\) and \(y\) approaches \(b\text{,}\) so that if \(f:\bbbr^2\rightarrow\bbbr\) is defined by \(f(x,y) = x\text{,}\) then
    \begin{equation*} \lim_{(x,y)\to(a,b)} f(x,y) = a \end{equation*}
    Similarly, if \(g:\bbbr^2\rightarrow\bbbr\) is defined by \(g(x,y) = y\text{,}\) then
    \begin{equation*} \lim_{(x,y)\to(a,b)} g(x,y) = b \end{equation*}
