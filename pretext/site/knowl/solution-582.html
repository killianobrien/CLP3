Solution 4.4.4.1.

The base case of \(\L{t^0}=\L{1}=1/s\) has already been proven. If \(\L{t^k} = 1/s^{k+1}\) then

\begin{align*} \L{t^{k+1}} \amp = \L{t \cdot t^k},\\ \amp = -\frac{d}{ds} \left ( \frac{1}{s^{k+1}} \right ),\\ \amp = \frac{1}{s^{k+2}}. \end{align*}

and so the result follows by induction on \(n\text{.}\)
