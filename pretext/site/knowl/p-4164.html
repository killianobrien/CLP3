The base case of \(\L{t^0}=\L{1}=1/s\) has already been proven. If \(\L{t^k} = k!/s^{k+1}\) then

\begin{align*} \L{t^{k+1}} \amp = \L{t \cdot t^k},\\ \amp = -\frac{d}{ds} \left ( \frac{k!}{s^{k+1}} \right ),\\ \amp = - \left (-(k+1) \frac{k!}{s^{k+2}} \right ) \\ \amp = \frac{(k+1)!}{s^{k+2}}. \end{align*}

and so the result follows by induction on \(n\text{.}\)
