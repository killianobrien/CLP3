Definition 5.5.1. Pointwise convergence.

A sequence of functions \(f_n:\mathbb{R} \to \mathbb{R}\) converges pointwise to a limit function \(f:\mathbb{R} \to \mathbb{R}\) if and only if for each argument \(a \in \mathbb{R}\) the sequence of values \(\left \{ f_n(a) \right \}_{n=1}^\infty\) converges to the limit value \(f(a)\text{,}\) i.e. for all \(a \in \mathbb{R}\) and for all \(\epsilon \gt 0\) there exists \(N \in \mathbb{N}\) such that \(|f_n(a) - f(a)| \lt \epsilon\text{.}\)
