We describe here a four-step method for solving a linear ODE of the form

\begin{equation*} a_n \frac{d^ny}{dt^n} + \dots + a_1 \frac{dy}{dt} + a_0y = g(t), \end{equation*}

by finding a suitable expression defining the unknown function \(y(t)\text{.}\) The method should work as long as the function \(g\) has a reasonable Laplace transform. Usually we shall have appropriate initial values for the function \(y\) and its derivatives so that there will be no undetermined constants in the solutions. The four steps are:

  1. Find the Laplace transform of the whole ODE. This will require using the transform of derivatives property from Theorem 4.4.5 to transform the left hand side and using the table of transforms, if necessary, to transform the function \(g\) on the right.

  2. Insert the initial conditions. This should supply actual values for the undetermined constants \(y_i\) from the left hand side.

  3. Solve the resulting equation for \(\bar{y}\text{,}\) the Laplace transform of \(y\text{.}\) This step should be straightforward and just relies on regular algebraic manipulation to find a formula in the variable \(s\) for \(\bar{y}\text{.}\)

  4. Find the inverse transform of \(\bar{y}\) to get the solution \(y(t)\) of the original ODE. This may well require some simplifying work do be done on \(\bar{y}\) first, so that we have it in a form that enables us to find its inverse transform using the table.
