Example 1.2.2.

Show that the Fourier series for the square wave function \(f\) defined by

\begin{equation*} f(x) = \begin{cases} 0 \amp \mbox{if } -\pi \lt x \lt 0 \\ 1 \amp \mbox{if } 0 \leq x \leq \pi . \end{cases} \end{equation*}

is given by

\begin{align*} f(x) \amp = \frac{1}{2} + \frac{2}{\pi} \left (\sin(x) + \frac{1}{3}\sin(3x) + \frac{1}{5} \sin(5x) + \dots \right ),\\ \amp = \frac{1}{2} + \frac{2}{\pi} \sum_{n=1}^\infty \frac{1}{2n-1} \sin \big((2n-1)x\big), \end{align*}

and verify that the series has the required value at the point of discontinuity \(x=0\text{.}\)
