In general a transform of a function, \(x(t)\) say, is a rule for associating a new function, \(X(s)\) say, to \(x\text{.}\) The new function depends on a new argument / variable \(s\text{.}\) The values of the original function \(x(t)\) are used somehow to define the new function \(X\text{.}\)
