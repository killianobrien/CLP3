Consider the transformation

\begin{equation*} t = \frac{\pi x}{T}. \end{equation*}

As the variable \(x\) ranges from \(-T\) to \(T\text{,}\) the variable \(t\) ranges from \(-\pi\) to \(\pi\text{.}\) We can then show that if \(f\) is a function of period \(2T\) (which is well-behaved as described in Fourier's theorem) then \(f\) has a Fourier series

\begin{equation*} f(x) = \frac{1}{2} a_0 + \sum_{n=1}^\infty a_n \cos \left ( \frac{nx\pi}{T}\right ) + \sum_{n=1}^\infty b_n \sin\left ( \frac{nx\pi}{T}\right ) , \end{equation*}

where the Fourier coefficients are given by

\begin{align*} a_0 \amp = \frac{1}{T} \int_{-T}^T f(x) \, dx,\\ a_n \amp = \frac{1}{T} \int_{-T}^T f(x) \cos\left ( \frac{nx\pi}{T}\right ) \, dx, \quad (n \geq 1)\\ b_n \amp = \frac{1}{T} \int_{-T}^T f(x) \sin\left ( \frac{nx\pi}{T}\right ) \, dx \quad (n \geq 1). \end{align*}
