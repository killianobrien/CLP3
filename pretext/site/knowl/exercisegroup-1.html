Exercise Group 1.1.1.1–4.

Exercises — Stage 1

1.

Describe the set of all points \((x,y,z)\) in \(\bbbr^3\) that satisfy

  1. \(\displaystyle x^2 +y^2+z^2= 2x-4y+4\)
  2. \(\displaystyle x^2 +y^2+z^2 \lt 2x-4y+4\)
2.

Describe and sketch the set of all points \((x,y)\) in \(\bbbr^2\) that satisfy

  1. \(\displaystyle x=y\)
  2. \(\displaystyle x+y=1\)
  3. \(\displaystyle x^2+y^2=4\)
  4. \(\displaystyle x^2+y^2=2y\)
  5. \(\displaystyle x^2+y^2 \lt 2y\)
3.

Describe the set of all points \((x,y,z)\) in \(\bbbr^3\) that satisfy the following conditions. Sketch the part of the set that is in the first octant.

  1. \(\displaystyle z = x\)
  2. \(\displaystyle x + y + z = 1\)
  3. \(\displaystyle x^2 + y^2 + z^2 = 4\)
  4. \(x^2 + y^2 + z^2 = 4\text{,}\) \(z = 1\)
  5. \(\displaystyle x^2+y^2=4\)
  6. \(\displaystyle z = x^2 + y^2\)
4.

Let \(A\) be the point \((2,1,3)\text{.}\)

  1. Find the distance from \(A\) to the \(xy\)-plane.
  2. Find the distance from \(A\) to the \(xz\)-plane.
  3. Find the distance from \(A\) to the point \((x,0,0)\) on the \(x\)-axis.
  4. Find the point on the \(x\)-axis that is closest to \(A\text{.}\)
  5. What is the distance from \(A\) to the \(x\)-axis?
