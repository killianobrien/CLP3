Section 1.4 Properties of the transform

We now discuss some important properties of the transform that follow from the definition 1.2.1 and examples of using them to find transforms of futher functions.

Theorem 1.4.1. Linearity of the transform.

The Laplace transform is linear, i.e. if \(f\) and \(g\) are two functions with Laplace transforms then for all \(\alpha, \beta \in \mathbb{R}\text{,}\)

\begin{equation*} \L{\alpha f + \beta g} = \alpha\L{f} + \beta \L{g}. \end{equation*}
Proof.

This follows directly from the linearity of integration.

Example 1.4.2. Appying linearity.

Find the Laplace transform \(\L{3t + 4e^{-2t}}\text{.}\)

Solution
\begin{align*} \L{3t + 4e^{-2t}} \amp = 3 \L{t} + 4 \L{e^{-2t}} , \\ \amp = \frac{3}{s^2} + \frac{4}{s+2}. \end{align*}

The next result links the derivative of a transform to the process of multiplying the original function by \(t\text{.}\)

Theorem 1.4.3. Derivative of the transform.

Let \(f\) be a function with transform \(F(s) = \L{f(t)}\text{,}\) then

\begin{equation*} \L{tf(t)} = - \frac{dF}{ds}. \end{equation*}
Proof.
\begin{align*} -\frac{dF}{ds} \amp = - \frac{d}{ds} \left ( \trans{f(t)} \right ) , \\ \amp = - \int_0^\infty \frac{\partial}{\partial s} \left ( e^{-st} f(t) \right ) \, dt ,\\ \amp = - \int_0^\infty -t e^{-st} f(t) \, dt \\ \amp = \trans{t f(t)}, \\ \amp = \L{tf(t)}. \end{align*}

In a previous example we found that \(\L{t}=\frac{1}{s}\text{.}\) Now we can apply theorem 1.4.3 to find \(\L{t^n}\) for any positive integer \(n\text{.}\)

Example 1.4.4. Transforms of polynomial terms.

For any integer \(n\geq 0\text{,}\)

\begin{equation*} \L{t^n} = \frac{1}{s^{n+1}} \end{equation*}
Solution

The base case of \(\L{t^0}=\L{1}=1/s\) has already been proven. If \(\L{t^k} = 1/s^{k+1}\) then

\begin{align*} \L{t^{k+1}} \amp = \L{t \cdot t^k},\\ \amp = -\frac{d}{ds} \left ( \frac{1}{s^{k+1}} \right ),\\ \amp = \frac{1}{s^{k+2}}. \end{align*}

and so the result follows by induction on \(n\text{.}\)

So now using linearity and example 1.4.4 we can find the transform of any poynomial in \(t\text{.}\)

The next property concerns taking the Laplace transform of the derivative of a function. This is a key property that enables the transform to be used as a tool to solve ODEs, as ODEs feature many derivatives of the unknown function that we wish to find.

Theorem 1.4.5. Transform of a derivative.

Let \(x\) be a function having a Laplace transform \(\L{x(t)}\text{.}\) Then

\begin{equation*} \L{\frac{dx}{dt}} = s \L{x(t)} - x(0). \end{equation*}
Proof.

This can be shown using integration by parts, as follows,

\begin{align*} \L{\frac{dx}{dt}} \amp =\trans{\frac{dx}{dt}} , \\ \amp = \left [ e^{-st} x(t) \right ]_0^\infty - \int_0^\infty -se^{-st} x(t) \, dt \\ \amp = -x(0) + s \trans{x(t)} ,\\ \amp = s \L{x(t)} - x(0), \end{align*}

as required.

The transforms of higher order derivatives can be obtained by iterating this result.

Corollary 1.4.6.

Let \(x\) be a function having a Laplace transform \(\L{x(t)}\) and let \(n \geq 1\text{.}\) Then

\begin{equation*} \L{\frac{d^n x}{dt^n}} = s^n \L{x(t)} - \sum_{i=0}^{n-1} s^{n-1 - i}x_i, \end{equation*}

where \(x_i\) denotes the initial value of the \(i\)th derivative of \(x\text{,}\) i.e.

\begin{equation*} x_i = \left. \frac{d^ix}{dt^i} \right |_{t=0}. \end{equation*}
Proof.

Use induction on \(n\text{,}\) base case provided by theorem 1.4.5.

Laplace transforms are invertible for most applications, and we will assume that this is the case. So we speak of the inverse Laplace transform of a function \(F(s)\) as being the function \(f(t)\) whose Laplace transform is \(F(s)\text{,}\) and we write

\begin{equation*} \Linv{F(s)} = f(t). \end{equation*}

So for example \(\Linv{\frac{1}{s^2}} = t\) and \(\Linv{\frac{1}{s-4}} = e^{4t}\text{.}\)

The inverse transform of an arbitrary function \(F(s)\) can be expressed/evaluated as a certain integral in the complex plane of the product of suitable exponential and \(F\text{.}\) However for our purposes we think of finding an inverse transform as an exercise in using the Laplace transform table in reverse, although in many cases it will be necessary to use the method of partial fractions as a precursor to this.
